

#  基于idea + mysql8.0 + javaGUI 开发的单词簿


  - 使用说明：
   1. 环境配置: 本项目基于idea+mysql8.0+javaGUI开发。
   2. 运行本项目前要先修改userlog.txt，将用户名和密码换成自己的。
   3. 运行log.java即可启动。


  - 功能说明：
   1. 单词增删改查
   2. 单词练习模式：中译英 、 英译中
   3. 联网查询（调用有道词典api）


  - 存在的问题:
   1. 背景音乐我们用的是绝对路径，需要时自己修改。
   2. 兼容性存在一定问题（exe文件可能会报异常，练习模式的中译英可能打不开）
